# NetNewsWire Theme

RSS 阅读器 [NetNewsWire](https://netnewswire.com/) 的自定义主题，使用了[霞鹜文楷](https://github.com/lxgw/LxgwWenKai)字体，适合阅读长文。

效果预览：

![reader.theme](https://raw.githubusercontent.com/mz1999/material/master/images/202402270933133.png)

## 安装字体

主题使用了[霞鹜文楷](https://github.com/lxgw/LxgwWenKai)和[Ysabeau](https://github.com/CatharsisFonts/Ysabeau)合并而成的[LXGW Bright](https://github.com/lxgw/LxgwBright)字体，兼顾中文和英语的优美体验。请前往**LXGW Bright**的[主页](https://github.com/lxgw/LxgwBright)下载并安装。

## 安装主题

双击 **reader.nnwtheme** 文件即可安装。然后在 NetNewsWire 的 Settings 中选择 reader 主题。

![settings](https://raw.githubusercontent.com/mz1999/material/master/images/202402270947230.png)
